# User Manual - New Features

## 1. Project Stage Rollback (Teacher)
**Scenario:** A student team accidentally passed a stage (e.g., "Proposal") but needs to make major changes.
**How to use:**
1. Go to **Course Detail** -> **Projects** tab.
2. Click **Manage Stages** on the target project.
3. In the stage list, if a stage is "Passed", click the **Rollback** button.
4. **Effect:** The stage becomes "Open" again, and any subsequent stages are locked.

## 2. Case Library Attachments (Teacher/Student)
**Scenario:** Teachers want to upload PDF/Docx files for a case study.
**How to use:**
1. **Teacher:**
   - Go to **Course Detail** -> **Case Library**.
   - Click **Create Case** or **Edit**.
   - In the dialog, use the **Upload** button to select files.
   - Click **Confirm** to save.
2. **Student:**
   - Go to **Case Library**.
   - Click **View** on a case.
   - Click the file name in the "Attachments" section to download.

## 3. Leader Transfer (Student)
**Scenario:** A team leader wants to step down and appoint another member as leader.
**How to use:**
1. Log in as the **Team Leader**.
2. Go to **Team Management**.
3. Click the **Transfer Leader** button.
4. Select the new leader from the team members list.
5. **Confirm**. You will become a regular member.

## 4. Force Leader Change (Teacher)
**Scenario:** A team leader has dropped out, and the teacher needs to assign a new leader.
**How to use:**
1. Go to **Course Detail** -> **Teams**.
2. Find the target team.
3. Click **Force Leader**.
4. Select the student to become the new leader.

## 5. Admin User Creation (Admin)
**Scenario:** Manually adding a student or teacher without import.
**How to use:**
1. Go to **User Management**.
2. Click **Create User**.
3. Select Role (Student/Teacher/Admin).
4. Fill in Username, Real Name, and Student/Teacher No.
5. Default password is `123456`.
